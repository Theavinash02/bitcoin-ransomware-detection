// What it does

Classifying ransomware activity on the Bitcoin network

Ransomware operators use Bitcoin to collect payments anonymously. By the time a victim pays, the address has already left a distinctive fingerprint in the transaction graph - patterns in how coins flow, loop, and accumulate.

ChainGuard analyses eight transaction-level features extracted from the Bitcoin blockchain and classifies each address as either benign (white) or belonging to one of 29 known ransomware families such as Locky, CryptoLocker, and WannaCry.

The model was trained as a Final Year Project at Anna University (B.Tech IT, 2024) and is being extended with Bayesian Network comparisons and SHAP explainability in v2.

// How it works

From raw transaction data
to a classification

01
📥
Raw blockchain data
Bitcoin transaction records for each address, including timing, value flows, and graph topology.
02
⚙️
Feature extraction
8 graph-derived features are computed per address: length, weight, count, looped, neighbors, income, year, day.
03
🌲
Random Forest
An ensemble of decision trees trained on the labelled BitcoinHeist dataset votes on the address classification.
04
🏷️
Label output
The address is labelled as benign (white) or assigned to a specific ransomware family with high confidence.
// Dataset & features

BitcoinHeist - UCI ML Repository

A dataset of Bitcoin addresses with labelled ransomware families, originally published by Farrukhjon Masudov et al. Each row represents one Bitcoin address with 8 computed graph features.

2.9M
Total addresses
29
Ransomware families
8
Input features
Feature Type Description
year int Year in which the transaction was recorded on the blockchain.
day int Day of the year (1–365) of the transaction.
length int Number of transactions in the chain leading to this address.
weight float Fraction of total BTC received from a single source address.
count int Total number of transactions associated with this address.
looped binary Whether coins were ever sent back to a previously seen source (1 = yes).
neighbors int Number of distinct addresses that transacted with this address.
income float Total BTC received by the address prior to this transaction.
